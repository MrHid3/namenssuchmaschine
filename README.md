# Programmierungsprojektsnamenssuchmaschine

Or, in other words, a program to help you find a creative name for a boring project! With the power of DeepL, it translates any English name to _every_ language <sup>(except for those using random scratches as alphabet)</sup>, allowing your _'boring'_ project to become _'makapakapoy'_! Doesn't that sound a little better? :P

## Usage
1. Install the latest version of Python on your machine
2. Create an `.env` file with your `DEEPL_API_KEY` in the same directory as the script
3. Install the dependencies with `pip install deepl dotenv`
4. Run the program with `python main.py`
